# 병원동행·처방전 관리 자동화 설계

작성자: 조경호 (시니어케어 3년 차)
목적: 어르신 병원동행·처방전 관리 업무를 자동화·정리하여, 요양보호사가 놓치지 않고 챙길 수 있도록 만든다.

---

## 1. 분해 (Decomposition)

업무를 시간 단위로 쪼개서 본다.

| 구분 | 내용 |
|---|---|
| 주별 | 이번 주에 병원 방문이 있는 어르신 확인, 이동 수단/동행자 확인 |
| 월별 | 이번 달 처방약 소진 예정일 확인, 재방문 필요 여부 정리 |
| 분기별 | 정기 검진(3개월 단위 혈액검사 등) 일정 확인 |
| 시간대별 | 당일 방문 시간, 진료과 접수 시간, 약국 방문 시간 |

## 2. 패턴 (Pattern)

- 어르신 대부분은 **정기적으로 같은 병원, 같은 진료과**를 방문한다.
- 처방약은 보통 **한 달(30일) 단위**로 나오고, 소진 시점이 곧 다음 방문 시점과 겹친다.
- 즉, "방문 주기"와 "약 떨어지는 주기"가 거의 같이 움직인다는 규칙성이 있다.

## 3. 추상화 (Abstraction)

어르신마다 병원도 다르고 질환도 다르지만, 이를 하나의 공통된 틀(카드)로 단순화할 수 있다. 이렇게 하면 몇 명을 담당하든 같은 방식으로 관리할 수 있다.

**"병원동행 이벤트" 카드**

| 항목 | 예시 |
|---|---|
| 고객명 | 김OO 어르신 |
| 병원 / 진료과 | OO내과 |
| 방문 주기 | 4주 |
| 마지막 방문일 | 2026-06-15 |
| 처방일수 | 30일 |
| 다음 방문 예정일 | 2026-07-13 |
| 약 소진 예정일 | 2026-07-15 |
| 동행 필요 여부 | 필요 (거동 불편) |
| 담당 요양보호사 | 본인 |
| 특이사항 | 혈압약 복용 중, 공복 채혈 필요 |

→ 어떤 어르신이든 이 8~9개 항목만 채우면 동일한 방식으로 스케줄을 관리할 수 있다.

## 4. 알고리즘 (Algorithm)

규칙 기반으로 자동 알림/스케줄링이 이루어지는 절차다. 별도 프로그램 개발 없이 **구글 캘린더 반복일정 + 알림 기능** 또는 **엑셀/구글시트 체크리스트**로 그대로 구현 가능하다.

1. **다음 방문 예정일 계산**: 마지막 방문일 + 방문 주기
2. **약 소진 예정일 계산**: 마지막 처방일 + 처방일수
3. **사전 알림 생성**: 소진 예정일 3~5일 전에 알림이 뜨도록 설정 (재방문 또는 약 리필 필요 신호)
4. **일정 병합**: 담당하는 모든 어르신의 알림을 하나의 주간/월간 캘린더에 모아서, 동행이 필요한 날짜에 미리 시간을 비워둔다.
5. **방문 완료 후 갱신**: 방문을 마치면 "마지막 방문일"과 "처방일수"를 새 값으로 업데이트하고, 다시 1번부터 반복한다.

---

## 5. 실제 적용 예시 (2주 앞 미리보기)

담당 어르신 2명을 예로 든 주간 체크리스트다.

| 날짜 | 어르신 | 할 일 | 동행 필요 |
|---|---|---|---|
| 7/13(월) | 김OO | OO내과 재방문 (혈압약 처방) | O |
| 7/15(수) | 박OO | 정형외과 약 소진 예정 → 리필 필요 알림 | X (보호자가 약국 방문) |
| 7/20(월) | 박OO | 정형외과 재방문 | O |

이렇게 표 하나만 보면 "이번 주에 누구를, 언제, 왜 챙겨야 하는지"가 한눈에 보인다.
